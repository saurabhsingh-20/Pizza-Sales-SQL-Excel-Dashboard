# 🍕 Pizza Sales Analysis Dashboard

## 📖 Overview

This project analyzes pizza sales data using **SQL** and **Microsoft Excel** to generate business insights and build an interactive sales dashboard.

The objective of this project is to understand customer purchasing behavior, identify sales trends, evaluate product performance, and present key business metrics through a dashboard.

---

## 🛠️ Tools Used

- SQL
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Design

---

## 📊 Key Performance Indicators (KPIs)

- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order

---

## 📈 Business Analysis Performed

- Daily Order Trend Analysis
- Hourly Sales Trend Analysis
- Sales by Pizza Category
- Sales by Pizza Size
- Top 5 Best-Selling Pizzas
- Bottom 5 Least-Selling Pizzas

---

## 📂 Files Included

- `Pizza_Sales_Dashboard.xlsx` – Interactive Excel dashboard
- `SQL_Queries.docx` – SQL queries used for data analysis

---

## 💡 Key Insights

- Identified peak sales days and busiest ordering hours.
- Compared revenue across pizza categories and sizes.
- Ranked products based on sales performance.
- Built an interactive dashboard to support business decision-making.
